## Project Overview 
This project analyzes coffee sales data using Microsoft Excel. The objective is to identify sales trends, store performance, customer purchasing behavior, and product performance through Pivot Tables, Pivot Charts, KPIs, and an interactive dashboard.

## Skills Used
-Excel Tables
-Data Cleaning
-Conditional Formatting
-IF, SUMIFS, COUNTA, UNIQUE etc.
-Pivot Tables
-Pivot Charts
-Slicers
-Dashboard Design
-Business Analysis
## KPIs
-Total Sales
-Total Orders
-Total Quantity Sold
-Average Order Value
-Average Unit Price
-Highest Unit Price
-Lowest Unit Price
-Number of Store Locations
## Business Insights 
-Coffee category generated the highest revenue.
-Astoria store recorded the highest sales.
-Weekday sales were higher than weekend sales.
-Sales increased steadily from January to June.
-The average order value is ₹4.69.
-Customers mostly purchased low-priced products in high volumes.
-Coffee products contributed the largest share of total quantity sold.

## Recommendations (optional but impressive)
-Increase inventory for top-selling coffee products.
-Promote weekend offers to improve weekend sales.
-Focus marketing on lower-performing stores.
-Introduce combo offers to increase average order value.
